# Proyecto Karate con Bootstrap

## Descripción
Proyecto web que muestra los distintos cinturones de Karate y sus técnicas requeridas usando Bootstrap 5. Cada cinturón tiene su propia página con katas, técnicas y vídeo demostrativo.

## Tecnologías
- HTML
- CSS
- Bootstrap 5
- JavaScript (opcional para interacción)

## Pasos de instalación y ejecución
1. Clonar el repositorio:
2. Abrir `index.html` en cualquier navegador.
3. Navegar por los distintos cinturones y la sección de contacto.

- Cada página incluye botones y vídeos de demostración.
